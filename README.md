## Go Boilerplate

Minimal starter template for a Go web service using Gin, MySQL, Redis, and Kafka.

### Quick start

```bash
git clone git@github.com:mamh1019/go-boilerplate.git
cd go-boilerplate
go mod tidy
go run ./...
```

### Environment

- `MYSQL_DSN`
- `REDIS_ADDR`
- `KAFKA_BROKERS`
